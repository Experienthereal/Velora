Requirements:

The user must have Clang installed on their system (Clang is required for compiling Velora source code to machine code).
If Clang is not installed, an error message should appear during the compilation step.
After installing Clang, the user should run Velorainstaller.exe (or VeloraInstaller on non-Windows systems) to complete the setup.
Encourage users to report any bugs they find while using Velora.
If users want to fix a bug themselves, they can send the fix to us.
Provide the CEO’s email address for bug reports and contributed fixes: andrejbaltes4@proton.me
Mention in a positive and welcoming way that the CEO loves receiving bug reports and community fixes.
